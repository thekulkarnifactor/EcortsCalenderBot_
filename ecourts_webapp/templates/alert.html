Usage Examples
JavaScript Usage:
// Simple alert
showAlert('Message here', 'success');

// With title
showAlert('Message', 'warning', 'Title');

// Confirmation
showConfirm('Delete this item?', 'danger')
  .then(confirmed => {
    if (confirmed) {
      // User clicked OK
    }
  });
Available Types:
  • success - Green success messages
  • danger - Red error messages
  • warning - Orange warning messages
  • info - Blue information messages
  • primary - Purple primary messages
